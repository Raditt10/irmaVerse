# Notification Sound Placeholder

To add notification sound:
1. Download or create a notification sound file (MP3 format recommended)
2. Save it as `notification.mp3` in this directory
3. Recommended: Short sound (0.5-2 seconds), not too loud

Free sound resources:
- https://pixabay.com/sound-effects/
- https://freesound.org/
- https://mixkit.co/free-sound-effects/notification/

The app will gracefully handle if this file is missing.
